Sobre a plataforma

A plataforma foi criada para facilitar a criação de uma playlist para a festa de final de 2º periodo

A plataforma está programada em HTML + CSS com um back-end de Flask (Python). Tem um login flow completamente custom com uma base de dados MongoDB. O front-end usa UIKit para o styling.
